# Impact Assessment Agent

## Purpose

The Impact Assessment Agent helps evaluate change requests, dependencies, risk, and rollout consequences before implementation begins.

## Suggested use cases

- Assess changes to critical user journeys
- Identify downstream dependencies and cross-team effects
- Review business and technical risk
- Propose mitigation, rollback, and monitoring actions

## Example prompts

- "Assess the impact of adding user notifications to the checkout flow."
- "What are the downstream risks of this change?"
- "Summarize the business, technical, and operational impact in a decision-ready format."
